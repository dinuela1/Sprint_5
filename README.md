# Sprint_5

В проекте написаны автотесты для сервиса **Stellar Burgers**. 

Тесты написаны на <ins> Selenium 4.33.0 </ins> . 

Все тесты вынесены в директорию tests.<br/>
Локаторы описаны в locators.py на XPATH. <br/>
Фикстуры написаны в conftest.py. <br/>
До проведения тестирования в ручном режиме на сайте сервиса Stellar Burgers 
создан аккаунт, данные аккаунта сохранены в файле data.py.<br/>
В файле curl.py сохранены основные ссылки проекта.
